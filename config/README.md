# config/

| File | Committed | Description |
|---|---|---|
| `figma.defaults.json` | Yes | Shared defaults merged into local config |
| `figma.json` | No | Generated by `npm run init` (gitignored) |
| `mcp.example.json` | Yes | Figma MCP server example for any editor |

## Initialize

```bash
npm run init -- --url "https://www.figma.com/design/…" --collections Primitives Semantic …
npm run config
```

See **`docs/setup.md`**.
