# config/

| File | Committed | Description |
|---|---|---|
| `figma.defaults.json` | Yes | Static MCP settings shared by all installs |
| `figma.json` | No | Generated at init — your file key, URL, collections |

Initialize via Cursor chat (paste Figma URL) or:

```bash
npm run init -- --url "https://www.figma.com/design/…" --collections Primitives Semantic …
```
